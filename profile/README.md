## О проекте
**Happ Decrypt** - небольшой проект, нацеленный на борьбу с проприетарным куском кода от FlyFrog LLC - Happ. Мы предоставляем инструменты для расшифровки happ://crypt ссылок, а также прокси-ссылки для использования happ-only ссылок в других клиентах.

---

### Поддерживаемые форматы
Мы предлагаем каждому пользователю функционал для расшифровки следующих ссылок:
- `happ://crypt/`
- `happ://crypt2/`
- `happ://crypt3/`
- `happ://crypt4/`
- `happ://crypt5/`
